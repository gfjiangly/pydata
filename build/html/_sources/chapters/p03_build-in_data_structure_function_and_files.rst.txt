====================================
第三章：内建数据结构，函数和文件

我们将从python重度使用的数据结构开始：元组、列表、字典和集合。
然后我们将讨论创建你自己重复使用的python函数。
最后，我们将关注python文件对象技术和与本地硬盘交互。

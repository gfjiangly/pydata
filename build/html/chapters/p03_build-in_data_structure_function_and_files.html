第三章：内建数据结构，函数和文件¶

这章讨论贯穿全书内嵌入python语言的能力。 虽然像pandas和numpy等附加库为大规模数据集增加了高级计算函数，但它们是被设计和python内建数据处理工具一起使用。

我们将从python重度使用的数据结构开始：元组、列表、字典和集合。 然后我们将讨论创建你自己重复使用的python函数。 最后，我们将关注python文件对象技术和与本地硬盘交互。
